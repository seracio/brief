# fulgur ⚡️

> vis library

## Install

```
yarn add react d3 @seracio/fulgur
```

## Principles

-   uses d3 & React
-   vectorized elements when possible
-   declarative (jsx)
-   inherited props via the React context
-   server side rendering & static (svg)

Fulgur is the lib we use to make some of the vis in our website.
